about my work

I am interested in the full stack.

A majority of my work is done in Java and Java Swing Components, and much of my code can be found on github. I primarily work out of eclipse or netbeans, and the command line is an integral part of my life. While working on side projects on top of coursework during the academic year is tough, I believe it is absolutely necessary for becoming a well-rounded developer. I enjoy building sites for others professionally, as well as in my free time to feed my artistic voids.

ALBA, INC.

This is my most current project that is currently being developed using an Agile Software Development Method. This site is being developed with two other teammate as a prototype site for manufacturers to be able to buy inventory from suppliers. The front-end is built in HTML/CSS and JS and will be run on a Apache Tomcat server using Java as its middle architecture. We also will have a functional database using a JDBC connection. This project is still in its early stages so be forewarned that many parts are not functional or responsive yet. Stay tuned as more rollout will come in each iteration!
